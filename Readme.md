Using:
- [IMDB API](https://www.imdb.com/interfaces/)
## Quickstart

### Prerequisites :page_with_curl:

- **Python 3.6+**
- **Pip3**
- **virtualenv**

### Installing :tongue:
```
$ pip install -r requirements.txt
```
### Usage :zap:
Download from  [Im]
Run **app.py** - You can run it in console ```python app```
Open in browser [localhost:8080](http://localhost:8080)
